# 🌿 React TSX Eğitim Kampı & Öğrenci Portalı

Bu proje, modern web teknolojileri kullanılarak geliştirilmiş, etkileşimli bir **Öğrenci Yönetim ve Eğitim Portalı** uygulamasıdır. Kullanıcıların ders içeriklerini takip edebileceği, ilerlemelerini anlık olarak izleyebileceği ve konu sonu testleri ile bilgilerini ölçebileceği bir platform sunar.

## ✨ Öne Çıkan Özellikler

* **Dinamik Ders Yönetimi:** Konu anlatımları ve kod örnekleri arasında hızlı geçiş.
* **İlerleme Takibi:** Tamamlanan derslerin anlık olarak genel ilerleme çubuğuna yansıması.
* **Etkileşimli Test Sistemi:** Yanlış/doğru analizini anında yapan, modern arayüzlü bilgi testleri.
* **Öğrenci Dashboard:** Kullanıcı bilgilerini ve sistem durumunu gösteren özel panel.
* **Modern Tasarım:** Tailwind CSS kullanılarak oluşturulmuş, ferah bahar yeşili (Mint & Green) teması.
* **Responsive Arayüz:** Mobil ve masaüstü cihazlarla tam uyumlu (Responsive) yapı.

## 🛠️ Kullanılan Teknolojiler

* **Framework:** [React](https://reactjs.org/) (Vite altyapısı ile)
* **Dil:** [TypeScript](https://www.typescriptlang.org/) (Tip güvenliği ve sürdürülebilir kod için)
* **Stil:** [Tailwind CSS](https://tailwindcss.com/) (Modern ve hızlı arayüz geliştirme)
* **İkonlar:** Lucide React & Emoji entegrasyonu
